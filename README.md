# api-test-mocha

### Get started
- Step1 : Clone or Download this repo
- Step2 : ```npm install```
- Step3 : ```sudo npm install -g mocha```
- Step4 : ```npm install -g newman```

### Test Result
- ```mocha```
- ```newman run test/api-test-mocha.postman_collection.json```
- ```sh test/curl/post.sh```
